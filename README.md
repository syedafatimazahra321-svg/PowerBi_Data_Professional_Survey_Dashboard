# PowerBi_Data_Professional_Survey_Dashboard

## Project Overview

This project analyzes the Data Professional Survey dataset using Power BI to explore insights about salaries, programming languages, education levels, and career paths in the data industry.

## Dashboard Insights

* Average salary by job title
* Perceived difficulty of entering data careers
* Most popular programming languages
* Education level distribution
* Survey respondents by country
* Average age and total survey participants

## Tools Used

* Power BI
* Excel
* Data Visualization
